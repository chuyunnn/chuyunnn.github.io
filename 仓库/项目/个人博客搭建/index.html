域名

想要个人域名可以：

  1. 买域名：去阿里、腾讯或 Namecheap 买一个
  2. 设置 DNS：在域名后台设置一条 CNAME 记录，指向 你的用户名.github.io
  3. GitHub 设置：在仓库的 Settings -> Pages 里，填写你的 Custom Domain
